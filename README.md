# NIMO HID Bootloader console client

Command line client to support the NIMO bootloaders

## Building

The bootloader requres the following prerequisites:
- libusb-1.0-0-dev
- libudev-dev

- Install prerequisites
- Run make
- The final output is ./hidBoot
